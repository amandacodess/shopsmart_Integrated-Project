⚡ DSA Implementation Details
Hashing
Product Hash Map
Every product is stored in a Python dictionary keyed by product ID. Lookup by ID is O(1) on average using open-addressing with collision chaining.
Avg Lookup: O(1)
Searching
Linear + Binary Search
Linear search scans the full product list by substring match. Binary search operates on a pre-sorted list to find products by exact rating in O(log n).
Linear: O(n) · Binary: O(log n)
Sorting
Timsort (Python .sort)
Products are dynamically sorted by rating, price, or popularity using Python's built-in Timsort — a hybrid merge/insertion sort optimised for real-world data.
O(n log n)
ML
TF-IDF + Cosine Similarity
Product names and categories are vectorised via TF-IDF. Cosine similarity between vectors identifies the closest content matches for recommendations.
Build: O(n·d) · Query: O(n)
