# 🛍️ Smart E-Commerce Product Recommendation System

## 📌 Overview

This project is a simple yet powerful **E-Commerce Web Application** that demonstrates how recommendation systems, data analytics, and basic business intelligence can be integrated into a real-world platform.

It consists of:

* 🛒 A **Customer Interface** for browsing products and viewing recommendations
* 📊 An **Admin Dashboard** for analyzing business data and insights

---

## 🎯 Features

### 👤 User Side (index.html)

* Browse products
* View product recommendations
* Clean and responsive UI
* Light blue + white themed design for better user experience

### 🧠 Recommendation System

* Suggests similar or relevant products
* Built using basic logic (can be extended to AI/ML models)

### 📊 Admin Dashboard (main.html)

* Visual insights into product data
* Basic analytics for business understanding
* Data representation for decision making

### 🧹 Data Handling

* Data cleaning concepts applied
* Structured dataset usage
* Foundation for future data analytics expansion

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Custom CSS (Light Blue + White Theme)
* **Logic:** JavaScript (for recommendations & interactivity)
* **Concepts Used:**

  * Data Structures & Algorithms (searching, sorting)
  * Data Analytics basics
  * Business Intelligence concepts

## 📁 Project Structure

```bash
shopsmart/
│── app.py                  # Main Flask application (routes + APIs)
│── generate_data.py        # Script to generate/prepare dataset
│── requirements.txt        # Python dependencies
│── README.md               # Project documentation

│── models/
│   └── clean_data.csv      # Processed dataset used for recommendations

│── templates/
│   ├── index.html          # User interface (shopping page)
│   └── main.html           # Admin dashboard

│── static/
│   ├── style.css           # Styling (UI theme)
│   ├── script.js           # Frontend logic & API calls
│   └── (assets if any)     # Images / additional files
```
